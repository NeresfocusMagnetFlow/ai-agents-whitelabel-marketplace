# AI Agents Marketplace - Mobile App

React Native mobile app for iOS and Android.

## Setup

```bash
cd mobile
npm install
npx expo start
```

## Features
- ✅ Browse agents
- ✅ View details
- ✅ Purchase with Stripe
- ✅ Manage licenses
- ✅ Profile management

## Build

```bash
# iOS
eas build --platform ios

# Android
eas build --platform android
```
